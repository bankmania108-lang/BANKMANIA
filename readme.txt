Advanced WordPress LMS plugin for competitive exam preparation targeting Banking, SSC, JAIIB, and other Indian competitive exams.

== Description ==

Smart Exam Platform (SEP) is an advanced WordPress LMS plugin specifically designed for Indian coaching institutes and competitive exam preparation. It offers a comprehensive solution for creating, managing, and delivering online exams with features tailored for Banking, SSC, JAIIB, CAIIB, UPSC, and other competitive exams.

= Key Features: =
* Advanced exam creation with multiple question types
* Support for 5 options (A, B, C, D, E) for complex questions
* Negative marking system
* Timer with pause/resume functionality
* Random question and option shuffling
* Detailed performance analytics
* Mobile-optimized interface
* Safe immersive mode (iOS-friendly)
* Result sharing via WhatsApp, Telegram, etc.
* Student dashboard with progress tracking
* Question bank management
* CSV import/export for questions and results
* Custom HTML hooks for promotions and lead generation
* n8n integration for automation
* All India ranking system

= Perfect for: =
* Banking exam preparation (IBPS, SBI, RBI, etc.)
* SSC exam preparation (CGL, CHSL, etc.)
* JAIIB/CAIIB exams
* UPSC preparation
* State government exams
* Coaching institutes
* Educational institutions

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/sep-smart-exam-platform` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Exam menu to create and manage your exams
4. Use shortcodes like [sep_exam id="123"] to display exams on pages/posts

== Frequently Asked Questions ==

= Can I create exams with 5 options instead of 4? =
Yes, our platform supports 5 options (A, B, C, D, E) for questions that require more choices.

= Is the immersive mode compatible with iOS? =
Yes, we use a safe immersive mode that works on iOS without violating security policies, unlike forced fullscreen APIs.

= Can I track student progress? =
Yes, we provide comprehensive dashboards with detailed analytics and progress tracking.

= Is it mobile-friendly? =
Absolutely! The platform is optimized for mobile devices and low-bandwidth connections.
